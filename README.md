# WeARM
### Expert ARM Assembly & QML Toolchain

**WeARM** est un environnement de développement ultra-performant dédié à l'apprentissage et à la maîtrise de l'assembleur ARM64. Conçu pour la vitesse, il intègre une gestion mémoire innovante et une interface moderne pour rendre le "bas niveau" accessible à tous.

---

## ✨ Points Forts

* **Performance O(1) :** Système d'Arène Linéaire (Slab Allocator) pour une allocation mémoire ultra-rapide sans fragmentation.
* **Localité Spatiale :** Optimisé pour le cache L1 du processeur afin de réduire les "Cache Miss".
* **Architecture Hybride :** Interface fluide en QML couplée à un moteur Core en C++.
* **Expertise IA :** Intégration prochaine de Gemini-IA pour l'assistance au code local.
